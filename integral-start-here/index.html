🧭 Integral · Data Start-Here

A guided tour of the data structures behind Integral's five systems — one step at a time.

Integral coordinates a community's production and decisions without money and without bosses. It runs on five interlocking systems. Each of these explainers walks you through one system by following a single real decision or flow, showing exactly which data objects are created and updated at every step — in plain language, with the technical schema always one click away.

New here? Start with the CDS — it's the system that governs everything else. Then follow the loop.

⚠️ Status: a proposed candidate — not ratified. Everything here (schemas, simulator, example records) is Phase-1 candidate work offered to the contributor community to review, challenge, and ratify. Nothing is adopted; the design questions are deliberately still open.
For the technically-inclined: the candidate schemas live in specs/ · open questions · schema exercise overview. Source: Developer Guide · White Paper.
